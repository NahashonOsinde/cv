LaTeX

  • blockdiagram.sty, a set of tikz macros to draw nice block diagrams, examples: [tex], [pdf]
  • Eps2pdf1Clic (in french), ajoute une commande eps2pdf dans le menu contextuel (sous windows) qui permet de transformer un eps en pdf sans ajouter de marge (nécessite Ghostscript).
  • perfectpdf.sty (in french), package pour générer un fichier pdf avec polices de caractères incorporées, hyperliens non soulignés et remplissant automatiquement les champs titre, auteur, sujet et mots-clés, exemple d'utilisation : [tex], [pdf]
  • examen.cls (in french), classe pour la réalisation de sujets d'examens, exemple d'utilisation : [tex], [pdf]
